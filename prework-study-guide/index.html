HTML

  • HTMl is the standard markup language for creating web pages.
  • It works together with CSS and JavaScript to create more well rounded projects, but can still be used on its own to create simple web pages.
  • By using elements, the HTML can tell the browser how and in what format to display content.
  • the header contains the general information about the webpage.
  • Alhtough the header element is not visable to the user, the body element is what can be shown.

CSS

  • Using CSS is the language used to further style an HTML Document.
  • CSS is unique in that it doesn't create any new elements (like HTML or JavaScript).
  • CSS can be used to polish the web page, giving it some depth and a better visual apperance.

Git

  • Git Hub is an internet hosting service for coders all around the world
  • using this platform one can utilize essential things needed in a coders work including repositorys, branches, commits, and pull requests.
  • one thing that makes Git Hub special is its easy collaboration with different coders
  • by using a centralized repository, users can upload and manage code all while making updated changes to their work.

JavaScript

  • Whereas HTML strictly handles the structure and content of a webpage, CSS defines the colors, aesthetics, visual layout, and organization.
  • We use CSS to define style rules for specific elements of an HTML page and format them using a wide range of style options.
  • Java Script executes its code in a specific order, known as Control Flow.
